# Jupyter Notebooks

This folder contains Jupyter notebooks for OSINT analysis and data verification.

## How to Use Jupyter with This Repository

### Option 1: Launch from Anaconda Navigator
1. Open **Anaconda Navigator** on your Mac
2. Click **Launch** under Jupyter Notebook or JupyterLab
3. In the Jupyter interface, navigate to: `/Users/malikosbourne/Documents/GitHub/osint-portfolio-Malik/notebooks/`
4. Create new notebooks or open existing ones

### Option 2: Launch from Terminal
1. Open Terminal
2. Navigate to this repository:
   ```bash
   cd /Users/malikosbourne/Documents/GitHub/osint-portfolio-Malik
   ```
3. Activate Anaconda (if needed):
   ```bash
   source ~/anaconda3/bin/activate
   # or
   conda activate base
   ```
4. Launch Jupyter:
   ```bash
   jupyter notebook notebooks/
   # or
   jupyter lab notebooks/
   ```

### Option 3: Launch from Anaconda Prompt
1. Open Anaconda Prompt (or Terminal with Anaconda activated)
2. Navigate to the repository folder
3. Run: `jupyter notebook` or `jupyter lab`

## Notebook Organization
- Create separate notebooks for each investigation or analysis
- Name them descriptively (e.g., `investigation-1-image-verification.ipynb`)
- Keep code well-documented with markdown cells
